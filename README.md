# blockhosts
## How to use
    git clone https://github.com/yuowo39/blockhosts.git
    cd blockhosts/
    go build blockhosts.go
    ./blockhosts tiktok-hosts.txt
## Arguments
    -f       Default: IPv4
    -s       IPv6
    -h       Default: Hosts(Windows, macOS, Linux)
    -d       Dnsmasq
    FilePath
